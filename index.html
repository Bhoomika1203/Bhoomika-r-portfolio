01 / 05

InsectNet

Developed InsectNet, an ESP32-based GSM-connected monitoring system with on-device TinyML pipelines for image segmentation, insect counting, and classification, enabling real-time field analysis for entomology research. Collaborated with Western Sydney University and GB Pant University of Agriculture and Technology to build deployable solutions for tracking insect populations in natural environments. Integrated AWS services (Lambda, EC2, S3) to create a scalable, low-latency pipeline for remote image processing, storage, and analytics. Managed Linux EC2 instances via SSH to ensure reliable deployment, continuous monitoring, and support for real-time data workflows.

TinyML AWS Lambda · EC2 · S3 ESP32 · GSM Image Segmentation
🦟
02 / 05

Neuro-Controlled Prosthetic Arm

Engineered a prosthetic arm controlled directly by EEG brainwave signals. A Random Forest classifier achieves 87% grasp detection accuracy, integrated with an Arduino embedded system to actuate servo motors with ~250ms response latency — a low-cost, portable assistive technology prototype.

EEG Signal Processing Machine Learning Arduino Embedded C/C++
🦾
03 / 05

Wearable Respiratory Monitor

Designed a wearable respiratory device using a 3-pin NTC thermistor for temperature-based breath detection. Implements real-time waveform analysis to calculate breaths per minute, with Over-the-Air update capabilities on the ESP32 for flexible remote healthcare deployment.

ESP32 OTA Updates Real-time DSP NTC Thermistor
🫁
04 / 05

OTA Communication with ESP32-S3 Eye

Developed a full IoT system using the ESP32-S3 Eye for wireless data transmission and remote monitoring. Enables remote device control and real-time data analysis through mobile integration, demonstrating deep expertise in IoT protocols and OTA data management.

ESP32-S3 Eye IoT Protocols Mobile Integration
📡
05 / 05

Neonatal Asphyxia Detection

Worked on automated classification of neonatal cry signals for asphyxia detection using AI/ML. Implemented an encoder–decoder architecture achieving 82% accuracy in distinguishing asphyxiated from non-asphyxiated cry signals. Utilised MATLAB for spectrogram-based visualisation and feature extraction for model development.

Deep Learning Signal Processing MATLAB Healthcare AI
